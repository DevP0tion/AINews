## AI/IT 뉴스
### Zig 프로젝트, AI 생성 기여 전면 금지 정책 공개
Zig 오픈소스 프로젝트는 이슈·PR·댓글 등 모든 기여 경로에서 LLM 생성 콘텐츠를 공식 금지했습니다. 프로젝트 측은 '기여자 육성이 코드 품질보다 우선'이라며, LLM 기여는 리뷰어의 시간을 낭비하고 기여자의 실질적 성장을 막는다고 설명합니다. 오픈소스 커뮤니티에서 AI 생성 기여의 적절성을 둘러싼 논쟁이 다시 주목받고 있습니다.
[원문](https://simonwillison.net/2026/Apr/30/zig-anti-ai/)

### PyTorch Lightning PyPI 패키지에서 공급망 악성코드 발견
AI 훈련 프레임워크 PyTorch Lightning의 PyPI 패키지 `lightning` 2.6.2~2.6.3 버전이 악성 코드로 오염됐습니다. 해당 코드는 import 시 자동 실행돼 인증 토큰·환경 변수·클라우드 시크릿을 탈취하고, Claude Code 및 VS Code에 후킹을 심어 재실행을 지속시킵니다. 영향을 받은 개발자는 자격증명 전체 교체와 저장소 감사를 즉시 시행해야 합니다.
[원문](https://semgrep.dev/blog/2026/malicious-dependency-in-pytorch-lightning-used-for-ai-training/)
